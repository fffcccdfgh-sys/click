# Support

Click is a local Android automation tool. Support is best-effort.

## Before Opening An Issue

Please check:

- [README.md](README.md) for setup and build instructions.
- [docs/normal-programming-guide.md](docs/normal-programming-guide.md) for
  ordinary Lua scripting.
- [docs/pvz2-calibration-variables.md](docs/pvz2-calibration-variables.md) for
  PVZ2-specific scripting and calibration variables.
- [PRIVACY.md](PRIVACY.md) and [SECURITY.md](SECURITY.md) for permission and
  safety notes.

## What To Include

For bug reports, include:

- App version or commit SHA.
- Android version and device model.
- Whether the issue happens in debug or release builds.
- Short reproduction steps.
- Non-sensitive logs if available.

Do not include signing keys, passwords, tokens, account data, private
screenshots, or scripts you do not have permission to share.

## Security Issues

For vulnerabilities, follow [SECURITY.md](SECURITY.md). Do not post sensitive
details in a public issue.
