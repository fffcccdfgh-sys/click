# PVZ2 校准教程

PVZ2 专用脚本依赖屏幕校准点位。不同设备分辨率、游戏 UI 缩放和系统导航栏状态都可能让坐标不同，所以第一次使用前建议按自己的设备重新校准。

## 校准前准备

1. 打开 PVZ2 并进入脚本需要使用的界面。
2. 保持横屏方向、分辨率、系统导航栏状态不变。
3. 打开 Click 的 PVZ2 悬浮面板。
4. 进入校准面板，按分类逐项校准。
5. 校准后回到 PVZ2 脚本页面，确认对应分类显示为已校准。

## 推荐校准顺序

1. 植物卡槽
2. 种植棋盘
3. 阳光相关
4. 能量豆相关
5. 神器相关
6. 黄瓜相关
7. 充值/刷新相关
8. 战斗文字和颜色识别区域
9. 开始战斗相关
10. 其他位置
11. 无尽补给相关

先校准常用点位，再校准无尽补给这类流程点位。这样脚本调试时更容易判断问题出在哪里。

## 每类校准大概点什么

- 植物卡槽：依次点第 1 到第 8 个植物卡槽中心。
- 种植棋盘：按提示点棋盘格子的代表位置，用来推算行列坐标。
- 阳光相关：点阳光数值、阳光拾取或脚本需要判断的位置。
- 能量豆相关：点能量豆图标、数量或使用相关位置。
- 神器相关：点神器按钮和需要判断的状态位置。
- 黄瓜相关：点黄瓜按钮或释放位置。
- 充值/刷新相关：点脚本需要识别或点击的刷新、确认、取消等位置。
- 战斗文字和颜色：框选或点选脚本需要 OCR/颜色判断的区域。
- 开始战斗相关：点开始战斗、确认、继续等入口位置。
- 其他位置：放一些不属于上面分类但脚本会用到的点位。
- 无尽补给相关：校准补给文字区域、能力卡区域、确认按钮、继续挑战等位置。

## 常见问题

- 脚本点偏了：重新校准对应分类，并确认横屏方向和导航栏状态没有变化。
- OCR 识别不稳定：重新框选文字区域，尽量只包含需要识别的文字。
- 换设备后失效：每台设备都需要重新校准。
- 游戏 UI 更新后失效：如果按钮或界面位置变了，需要重新校准受影响的分类。
